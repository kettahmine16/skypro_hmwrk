Git

  • git status - проверка статуса репозитория, вывод списка добавленных, изменённых, или удалённых файлов.
  • git add - выбор конкретного файла для включения в будущий коммит, добавление его в индекс.
  • git commit - формирование нового коммита с комментарием, описывающим суть внесённых правок.
  • git clone - скачивание удалённого репозитория и создание его локальной копии.
  • git push - отправка локальных изменений в удалённый репозиторий.
